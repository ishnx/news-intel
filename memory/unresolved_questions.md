## What This File Does

Every time the system analyzes a story, it generates follow-up questions —
things it doesn't know that could significantly change the analysis if known.

This file accumulates those questions. The system uses them to:
1. Prime attention when scanning future news
2. Generate relevant context for related analyses
3. Track our knowledge gaps

**When a question is answered by future news, `update_memory.py` moves it to
the "Resolved" section and adds the answer to master_memory.md.**
