# Master Memory
## Accumulated Intelligence Facts and Insights

---

## What This File Does

This is the system's long-term fact repository.
Every concrete, verifiable piece of intelligence gets stored here.

The `update_memory.py` script appends to this file after each analysis run.
You can also add entries manually.

**Format:** `- **YYYY-MM-DD** [story-slug]: The insight or fact`

---

## Instructions for Reading This File

- Entries are in chronological order (newest at bottom of each section)
- Each entry references the source story in [brackets]
- Use Ctrl+F / Cmd+F to search for specific topics
- Outdated facts should be struck through with ~~text~~ notation
