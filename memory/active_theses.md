# Active Theses
## Strategic Beliefs Under Active Monitoring

---

## What This File Does

This file tracks the system's active strategic theses — high-conviction beliefs
about how the world is evolving that deserve ongoing monitoring.

Each thesis:
- States a clear, falsifiable claim
- Has an evidence log that grows over time
- Has a conviction level (low/medium/high) that changes with evidence
- Gets closed when resolved or superseded

The `update_memory.py` script updates these theses after each analysis.

---

## How to Read Conviction Levels
- **High**: Multiple strong confirming datapoints, clear mechanism
- **Medium**: Some confirming evidence, mechanism plausible but unproven
- **Low**: Interesting hypothesis, minimal evidence, actively watching
