## What This File Does

This file tracks how the system's worldview (config/worldview.md) evolves.
After each significant analysis run, `update_memory.py` appends a summary
of what changed, what was confirmed, and what was challenged.

This is the "intellectual autobiography" of the intelligence system.
Reading it backward tells the story of how the system's understanding developed.
