# Recurring Patterns
## Dynamics That Repeat Across Contexts

---

## What This File Does

This file tracks patterns that appear repeatedly across different stories and domains.
A pattern becomes "established" when it has been observed 3+ times.

Patterns are the highest-order abstractions in the memory system.
They represent durable dynamics that can inform forward-looking analysis.

**The `update_memory.py` script increments pattern counts and adds new instances.**

---

## How to Use This

When analyzing a new story, check if it exemplifies any of these patterns.
The system automatically checks — but you can also apply them manually.

## Pattern Template

When adding a new pattern:

```
## [Pattern Name]
**First Observed:** YYYY-MM-DD
**Count:** N
**Status:** Emerging / Established / Superseded
**Domain:** Technology / Geopolitics / Finance / Cross-cutting

### Description
What the pattern is and why it recurs.

### Mechanism
Why this pattern keeps appearing (the underlying causal mechanism).

### Instances
List of specific examples with dates and story references.

### Predictive Value
What future situations this pattern helps anticipate.
```

## System-Initialized Hypotheses
*(Initial pattern hypotheses to watch for — not yet confirmed as recurring)*

**Hypothesis 1: Regulatory Lag**
Technology consistently advances faster than regulation, creating windows of
unconstrained adoption followed by abrupt regulatory interventions.
*Watch for: AI, crypto, biotech, space*

**Hypothesis 2: Geopolitical Tech Nationalism**
Countries systematically prioritize domestic technology champions over efficiency
when strategic industries are involved.
*Watch for: semiconductors, AI, space, defense tech*

**Hypothesis 3: Capital Concentration Before Democratization**
New technology platforms concentrate returns in early movers, then gradually
democratize access as competition and regulation catch up.
*Watch for: cloud, AI, genomics*

**Hypothesis 4: India Systemic Underestimation**
Global consensus systematically underestimates India's growth trajectory,
creating recurring surprise at actual performance data.
*Watch for: GDP data, FDI flows, tech sector growth*
