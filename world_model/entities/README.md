# World Model — Subfolder

This folder contains entity files auto-generated and maintained by the system.

## File Format

Each `.md` file covers one entity:
- **Overview**: Current status and key facts
- **Timeline**: Chronological development log  
- **Connections**: Relationships to other entities
- **Open Questions**: What we don't know but want to know

Files are created automatically when an entity first appears in an analysis.
You can also create files manually for entities you want to track proactively.

## Editing

Edit freely — add context, relationships, historical notes.
The system will continue appending to the timeline section.
