# /intelligence/daily — Daily Intelligence Digest

This folder's `index.html` shows today's intelligence reports.
It is regenerated automatically by `build_indexes.py` after each pipeline run.

## Access

- **Live:** `https://[username].github.io/[repo]/intelligence/daily/`
- **Local:** Open `index.html` in your browser

## Contents

Shows all reports analyzed today, ordered by intelligence score.
Clicking any report card opens the full analysis.

## Automation

This index is rebuilt automatically after every pipeline run.
No manual action needed.
