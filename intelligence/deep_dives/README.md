# /intelligence/deep_dives — Long-Form Research

This folder contains deep-dive analyses — longer-form research on specific topics.

## What is a Deep Dive?

A deep dive is different from a regular report:
- Covers a theme or topic over multiple stories
- Synthesizes multiple data points into a coherent narrative
- Typically 3-5x longer than a regular story analysis
- Often produced on a weekly or monthly cadence

## How to Request a Deep Dive

1. Create `staging/priority_override.json` with multiple related articles
2. Trigger a manual GitHub Actions run
3. OR: Edit `config/schedules.yaml` to add a weekly deep-dive run
